## Images Directory

This directory should contain images for the website. Here's what you need to add:

### Required Images:

1. **gergana.jpg** - Professional portrait of Gergana Nikolova
   - Recommended size: 500x600px
   - Use for the About page

2. **wine-hero.jpg** - Background image for the hero section (homepage)
   - Recommended size: 1920x1000px
   - Should show wine bottles, vineyard, or elegant wine setting

3. **wine-consultation.jpg** - Image for Wine Consultation service
   - Recommended size: 600x400px
   - Can show wine bottles, wine glass, or consultation setting

4. **wine-tasting.jpg** - Image for Wine Tasting Notes service
   - Recommended size: 600x400px
   - Can show wine tasting notes, wine glass, or tasting event

### Where to Find Images:

- **Free Stock Photos**: Unsplash, Pexels, Pixabay (search for "wine", "sommelier", "wine tasting")
- **Professional Photos**: If you have personal photos of Gergana, use those instead
- **Wine Images**: Search for "wine bottles", "wine glass", "vineyard"

### How to Add Images:

1. Save your chosen images in this folder
2. Rename them to match the filenames above
3. The website will automatically display them

### Important Notes:

- JPEG format is recommended for best quality and smaller file sizes
- Make sure images are optimized for web (not too large)
- The website uses placeholder backgrounds if images are missing
